起動コマンド例


## コンテナ起動例（rootless想定・ポート3000で公開）

```bash
podman run -d \
  --name open-webui \
  -p 3000:8080 \
  -v open-webui:/app/backend/data \
  -e OLLAMA_BASE_URL=http://host.containers.internal:11434 \
  --restart=always \
  ghcr.io/open-webui/open-webui:main

```

- GPUサポート: ``` --gpus all ```
- Single-User Mode (Disabling Login): ``` -e WEBUI_AUTH=False ```


## ヒント
- 必要に応じて、Windowsの環境変数で OLLAMA_HOST=0.0.0.0:11434 を設定。
